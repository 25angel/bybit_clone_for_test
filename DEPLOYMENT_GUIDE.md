# Руководство по установке приложения на реальное устройство

## Для iOS устройств (iPhone/iPad)

### Шаг 1: Подготовка устройства

1. **Подключите устройство к Mac через USB кабель**
   - Используйте оригинальный кабель Apple или сертифицированный кабель
   - Убедитесь, что кабель поддерживает передачу данных (не только зарядку)

2. **Разблокируйте устройство**
   - Разблокируйте iPhone/iPad с помощью Face ID, Touch ID или пароля

3. **Доверьтесь компьютеру**
   - На устройстве появится запрос "Доверять этому компьютеру?"
   - Нажмите **"Довериться"**

4. **Включите режим разработчика (для iOS 16+)**
   - Настройки → Конфиденциальность и безопасность → Режим разработчика
   - Включите переключатель
   - Перезагрузите устройство при запросе

### Шаг 2: Настройка в Xcode

1. **Откройте Xcode**
   ```bash
   open /Users/skyapathy/bybit/ios/Runner.xcworkspace
   ```

2. **Спарьте устройство**
   - В Xcode: Window → Devices and Simulators (или `Cmd + Shift + 2`)
   - Найдите ваше устройство в списке слева
   - Если устройство показывает "Unpaired", нажмите "Use for Development"
   - На устройстве подтвердите запрос на доверие

3. **Выберите команду разработчика**
   - В Xcode выберите проект Runner
   - Перейдите в Signing & Capabilities
   - Выберите вашу команду разработчика (Team)
   - Если команды нет, создайте Apple ID в настройках Xcode

### Шаг 3: Установка приложения

**Вариант 1: Через Flutter CLI (рекомендуется)**

```bash
# Перейдите в директорию проекта
cd /Users/skyapathy/bybit

# Проверьте подключенные устройства
flutter devices

# Запустите приложение на устройстве
flutter run -d <device-id>
```

**Вариант 2: Через Xcode**

1. Откройте `ios/Runner.xcworkspace` в Xcode
2. Выберите ваше устройство в верхней панели (рядом с кнопкой Run)
3. Нажмите кнопку Run (▶️) или `Cmd + R`

### Шаг 4: Разрешения на устройстве

При первом запуске на устройстве может потребоваться:
- Разрешить установку приложения (Настройки → Основные → VPN и управление устройством)
- Доверить сертификат разработчика

## Для Android устройств

> **Примечание:** В данный момент проект не содержит конфигурацию Android. Если нужно добавить поддержку Android, выполните:
> ```bash
> flutter create --platforms=android .
> ```

### Если Android уже настроен:

1. **Включите режим разработчика**
   - Настройки → О телефоне → Нажмите 7 раз на "Номер сборки"

2. **Включите USB отладку**
   - Настройки → Для разработчиков → USB отладка

3. **Подключите устройство**
   ```bash
   flutter devices
   flutter run
   ```

## Устранение проблем

### Устройство не определяется

1. **Проверьте кабель**
   - Попробуйте другой USB кабель
   - Убедитесь, что порт USB работает

2. **Перезапустите службы**
   ```bash
   # Для iOS
   killall -9 usbmuxd
   # Подождите несколько секунд, служба перезапустится автоматически
   ```

3. **Проверьте в Xcode**
   - Window → Devices and Simulators
   - Убедитесь, что устройство видно и спарено

### Ошибка "Device is not available because it is unpaired"

1. Откройте Xcode
2. Window → Devices and Simulators
3. Выберите устройство и нажмите "Use for Development"
4. Подтвердите на устройстве

### Ошибка подписи кода (Code Signing)

1. Откройте `ios/Runner.xcworkspace` в Xcode
2. Выберите проект Runner → Signing & Capabilities
3. Выберите вашу команду разработчика
4. Если команды нет:
   - Xcode → Settings → Accounts
   - Добавьте ваш Apple ID
   - Вернитесь в Signing & Capabilities и выберите команду

## Полезные команды

```bash
# Список всех устройств
flutter devices

# Запуск на конкретном устройстве
flutter run -d <device-id>

# Сборка для установки (iOS)
flutter build ios

# Сборка для установки (Android)
flutter build apk

# Очистка проекта
flutter clean
flutter pub get
```

## Дополнительная информация

- Для установки на устройства других пользователей потребуется Ad Hoc или App Store дистрибуция
- Для тестирования на устройствах без компьютера используйте TestFlight (iOS) или Firebase App Distribution (Android)
