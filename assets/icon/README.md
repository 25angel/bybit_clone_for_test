# Инструкция по установке иконки приложения

## Шаги для установки иконки Bybit:

1. **Подготовьте иконку:**
   - Создайте PNG изображение иконки размером **1024x1024 пикселей**
   - Иконка должна быть квадратной
   - Сохраните её как `app_icon.png` в этой папке (`assets/icon/app_icon.png`)

2. **Сгенерируйте иконки:**
   После того как вы поместите файл `app_icon.png` в эту папку, выполните команды:
   
   ```bash
   flutter pub get
   flutter pub run flutter_launcher_icons
   ```

3. **Пересоберите приложение:**
   ```bash
   flutter clean
   flutter build ios
   ```

## Требования к иконке:
- Формат: PNG
- Размер: 1024x1024 пикселей
- Квадратная форма
- Без прозрачности (alpha канал будет удален автоматически)

После выполнения этих шагов, иконка приложения будет установлена для iOS!
